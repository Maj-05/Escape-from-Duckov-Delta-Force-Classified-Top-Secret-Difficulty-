F2菜单中 隐藏分系统 隐藏分配置用来修改自定义

预设了三条隐藏分设置：隐藏分越高爆率越高（不勾选就是越低） ，隐藏分越高难度越高（不勾选越低） AI视野随分数增加（不勾选减少）

隐藏分默认1000分 最低0 最高 2000

击杀敌人：+2分（可调整）范围 -20 ~ 20

撤离成功：+100分（可调整）范围 -200 ~ 200

撤离失败：-400分（可调整）范围 -400 ~ 400

爆率调整：1000分基础，可以在隐藏分配置界面调整隐藏分对爆率的影响 增加 or 减少 爆率调整强度影响每次 获得/减少 隐藏分时 对爆率的影响大小（难度调整强度同理）

倍率范围配置

最低爆率倍率 (Min Loot Multiplier)

这无论隐藏分多么低，爆率倍率都不会低于这个值。 例如，设置为0.10倍，爆率最低也不会低于基础爆率的0.10倍（即10%）。

最高爆率倍率 (Max Loot Multiplier)

无论隐藏分多么高，爆率倍率都不会超过这个值。 例如，设置为3.00倍，爆率最高也不会超过基础爆率的3.00倍（即300%）。

最低/最高 难度倍率 (Min/Max Difficulty Multiplier)

无论隐藏分多么 低/高，AI难度倍率都不会 低/高 于这个值。 例如，设置为 0.50/2 倍，AI难度最 低/高 也不会低于基础难度的0.50倍（即50%）/2.00倍（即200%）。

最低视野倍率 (Min Vision Multiplier)

无论隐藏分多么低，AI视野倍率都不会低于这个值。 例如，设置为0.50倍，AI视野角度最低也不会低于基础视野角度的0.50倍（即50%）。
